1. Z Distribution (Standard Normal Distribution)

Definition

The standard normal distribution is a special case of the normal distribution with mean Œº = 0 and standard deviation œÉ = 1.

\[f(z) = \frac{1}{\sqrt{2\pi}} e^{-\frac{z^2}{2}}, \quad -\infty < z < \infty\]

\[Z = \frac{X - \mu}{\sigma}\]

Variance: 1
Standard Deviation: 1
Skewness: 0 (symmetric)
Kurtosis: 3 (mesokurtic)

\[Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}\]

\[Z = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}}\]

Z-Test for Proportions:

\[Z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}\]

2. t Distribution (Student's t-Distribution)

Definition

The t-distribution is used when the population standard deviation is unknown and must be estimated from the sample. It was developed by William Sealy Gosset under the pseudonym "Student".

Probability Density Function (PDF):

\[f(t) = \frac{\Gamma(\frac{\nu+1}{2})}{\sqrt{\nu\pi}\Gamma(\frac{\nu}{2})} \left(1 + \frac{t^2}{\nu}\right)^{-\frac{\nu+1}{2}}\]

where ŒΩ = degrees of freedom, Œì = Gamma function

Mean: 0 (for ŒΩ > 1)
Variance: ŒΩ/(ŒΩ-2) for ŒΩ > 2
Skewness: 0 (symmetric)
Kurtosis: 6/(ŒΩ-4) + 3 for ŒΩ > 4
Range: (-‚àû, +‚àû)
Parameter: ŒΩ (degrees of freedom)
Key Characteristics:
  • Bell-shaped and symmetric like normal distribution
  • Heavier tails than normal distribution
  • As ŒΩ ‚Üí ‚àû, t-distribution ‚Üí Z-distribution
  • Lower peak (more spread out) than Z for small df

One-Sample t-Test (œÉ unknown):

\[t = \frac{\bar{X} - \mu_0}{s / \sqrt{n}}\]

df = n - 1

Two-Sample t-Test (Equal Variances):

\[t = \frac{\bar{X}_1 - \bar{X}_2}{s_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}\]

where pooled variance: \(s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}\)

df = n‚ÇÅ + n‚ÇÇ - 2

\[t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}\]

df ‚âà \(\frac{\left(\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}\right)^2}{\frac{(s_1^2/n_1)^2}{n_1-1} + \frac{(s_2^2/n_2)^2}{n_2-1}}\) (Welch-Satterthwaite)

Paired t-Test:

\[t = \frac{\bar{d}}{s_d / \sqrt{n}}\]

where \(\bar{d}\) = mean of differences, s_d = std dev of differences

df = n - 1

When to Use t-Test

‚úì Use t-Test when:
  • Population standard deviation (œÉ) is unknown

\[\chi^2 = \frac{(n-1)s^2}{\sigma_0^2}\]

2. Chi-Square Goodness-of-Fit Test:

\[\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}\]

\[\chi^2 = \sum_{i=1}^{r} \sum_{j=1}^{c} \frac{(O_{ij} - E_{ij})^2}{E_{ij}}\]

df = (r - 1)(c - 1)

Use: Testing if two categorical variables are independent

Same formula as independence test

df = (r - 1)(c - 1)

Use: Testing if multiple populations have same distribution

Definition

The F-distribution is the ratio of two independent chi-square variables divided by their degrees of freedom.

\[SSW = \sum_{i=1}^k \sum_{j=1}^{n_i} (X_{ij} - \bar{X}_i)^2\]
